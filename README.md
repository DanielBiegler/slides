# slides

My presentations, built with [@danielbiegler/react-slides](https://github.com/danielbiegler/bieglers-react-slides).

## New presentation

```bash
./scaffold.sh <name>
```

## Run a presentation

```bash
cd presentations/<name> && bun run dev
```

## Build all presentations

```bash
bun run build
```

## Build a single presentation

```bash
cd presentations/<name> && bun run build
```
